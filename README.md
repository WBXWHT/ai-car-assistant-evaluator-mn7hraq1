# ai-car-assistant-evaluator

智能购车助手AI评测模拟系统

项目经历：我曾参与“智能购车助手AI评测体系构建”项目。针对购车决策场景，我基于GPT-4和国内大模型设计了自动化评测方案，通过构建包含2000+真实用户query的测试集，结合意图识别准确率与用户满意度设计分层评估指标。我负责设计AB实验对比不同Prompt策略的效果，最终将核心场景的意图理解准确率从78%提升至92%，并通过SQL进行数据归因分析输出迭代建议。项目推动建立了包含自动化评测、样本合成与双盲

## 运行
```bash
pip install -r requirements.txt
python main.py
```
